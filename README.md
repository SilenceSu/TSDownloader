#TSDowloader

练习项目-某听书网站下载器

## 环境准备
 - ###  安装chromedriver到path
    ``` 
    https://chromedriver.chromium.org/home
    ```
 - ### curl命令
 - ### java 1.8+
 - ### jar包

## 使用方式
  ### java -jar -proxy ip:prot -o downloadPath -url pageUrl
   - proxy 使用代理，因为会被屏蔽。所以非常建议用代理
   - downloadPath 下载目录，必须是绝对路径
   - pageUrl 直接去网站上去复制目录页地址
